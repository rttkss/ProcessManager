# 🚀 Process Manager Pro

![Main Window](screenshots/main.png)

## 📌 Описание проекта

**Process Manager Pro** - продвинутый диспетчер задач Windows с расширенным функционалом управления процессами, потоками и привязкой к ядрам процессора.

Проект выполнен в рамках лабораторной работы по курсу "Операционные системы".

---

## ✨ Основные возможности

### 🎯 Управление процессами
- Просмотр всех запущенных процессов в реальном времени
- Завершение процессов (Kill Process)
- Изменение приоритета выполнения процессов
- Автоматическое обновление списка каждые 5 секунд

### 🔍 Поиск и фильтрация
- Быстрый поиск процессов по имени
- Фильтр "Только с GUI" - показывает только программы с интерфейсом
- Фильтр "Системные процессы" - отображает только системные службы Windows

### 🧵 Работа с потоками
- Просмотр всех потоков выбранного процесса
- Отображение приоритета потоков
- Состояние потоков (Running, Wait, Terminated)
- Причина ожидания потока (WaitReason)

### ⚙️ CPU Affinity (Привязка к ядрам)
- Выбор конкретных ядер процессора для выполнения процесса
- Визуальное отображение маски ядер в BIN и HEX форматах
- Применение настроек в реальном времени

### 🌳 Дерево процессов
- Иерархическая структура процессов (родитель → дочерние)
- Отображение количества дочерних процессов
- Визуальная индикация высокоприоритетных процессов

### 📊 Мониторинг системы
- Отображение использования памяти каждым процессом
- CPU Time - время работы процессора на процесс
- Время запуска процесса
- Количество потоков
- Общая загрузка CPU системы
- Использование памяти приложением

## 🛠️ Технологии и инструменты

- **Язык программирования:** C# 7.0+
- **Фреймворк:** .NET Framework 4.7.2
- **UI:** WPF (Windows Presentation Foundation)
- **IDE:** Visual Studio 2022 Community
- **Система контроля версий:** Git + GitHub
- **ОС:** Windows 10/11

### Используемые библиотеки и API
- `System.Diagnostics` - работа с процессами и потоками
- `System.Runtime.InteropServices` - WinAPI для получения родительских процессов
- `System.Windows` - WPF интерфейс
- `ntdll.dll` - NtQueryInformationProcess для построения дерева

---

## 📋 Системные требования

### Минимальные:
- Windows 10 (1809) / Windows 11
- .NET Framework 4.7.2 или выше
- 2 ГБ RAM
- Права администратора (для полного доступа к системным процессам)

### Рекомендуемые:
- Windows 11
- 4 ГБ RAM
- Многоядерный процессор (для тестирования CPU Affinity)
